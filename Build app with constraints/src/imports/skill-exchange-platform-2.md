Develop a full-stack web application called “Skill Exchange Platform using Agentic AI.” The platform should enable users to teach and learn skills through a peer-to-peer exchange system without financial transactions. The system should intelligently connect users based on their skills, interests, and learning goals while building a collaborative knowledge-sharing community.

Additionally, the platform must include a skill verification mechanism, where users must submit certifications or proof of expertise before they are allowed to teach a skill. This ensures credibility and maintains the quality of knowledge shared on the platform.

1. User Authentication System

Implement a secure authentication system where users can:

Register using name, email, password, and profile details

Login and logout securely

Reset password if forgotten

Securely store credentials in the database

Enable session management and authentication security

2. Certification-Based Skill Verification

Before a user can teach a skill, the platform must verify their expertise.

Features include:

Users upload certifications, course completion documents, or portfolio proof

Admin or AI system verifies the certification

Once verified, the user receives a Verified Skill Badge

Only verified users are allowed to teach skills on the platform

Unverified users can still learn skills but cannot teach

This ensures trust, credibility, and quality learning experiences.

3. User Profile Management

Each user should have a dynamic personal profile containing:

Basic information (name, bio, interests)

Skills they can teach

Skills they want to learn

Skill proficiency level

Uploaded certifications

Verified skill badges

Profile picture and description

Ability to edit profile information

Add multiple skills they can teach

Provide a search and filter system to easily discover skills.

5. AI-Based Skill Matching (Agentic AI)

Implement an intelligent Agentic AI recommendation system that:

Analyzes user profiles, verified skills, and interests

Matches users with complementary skills

Suggests potential skill exchange partners

Recommends new skills to learn

Continuously improves recommendations using interaction data

6. Partner Discovery System

Users should be able to:

View AI-recommended partners
